How to add quotation details by importing parts information?

 

Quotation details can be imported from CSV file. User has already created a (CSV) file of requested parts along with requested quantities. Now import this file on quotation detail screen which will populate detail grid with all parts and quantities.

 

Follow the below steps to import parts list from a CSV file.

 

1)  Press the Browse button and select a CSV file that contains Parts List.

 

 

2)  Press Import button. Parts from CSV file are imported and displayed in list area.

 

 

 

3)  Press Calculate button. Listed parts are allocated and costs are calculated.

 

 

4)  Now press the Update button to save the added parts detail. A success message is displayed.
