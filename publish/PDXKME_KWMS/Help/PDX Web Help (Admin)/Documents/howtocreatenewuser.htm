How to create new user?

On Create New User screen:

1.  Enter User ID, User Name, Password, and Email ID of the user.

2.  Select user Role and enter Expiry Date of user account.

3.  Select DB Code from the dropdown. Assign DB Code(s) field will show codes for selected DB Code. Select code(s).

 

 

4.  Available Rights section shows features and functionalities that Parts System provides for each module. Select a module tab. Now mark and move features in Allocated Rights section to allow their access for the new user.

 

 

5.  Once done, press the Save button. New user will be created.
