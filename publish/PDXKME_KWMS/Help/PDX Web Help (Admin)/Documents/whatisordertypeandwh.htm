What is order type? And what is the purpose of choosing an order type?

Most important thing in creating a quotation is the “Order Type”. Pricing and allocation varies on the basis of order types. E.g. if user is placing quotation of SO (Stock Order) type then he will get lower prices as compared to EO (Emergency Order) type. Similarly EO (Emergency Order) gets high priority in terms of stock allocation. When order type is selected few of the fields are populated like order probability, usance, transportation, delivery term, payment term and billing rates.
