What does calculate button do?

When user enters a part in detail grid along with requested quantity and presses “Calculate” button then following operations are performed.

 

1.  Processed part number against requested part with its description is displayed.

2.  Accepted quantity against requested quantity is displayed.

3.  Unit Price, Price Amount, Selling Price, Discount Per Item, Tax Amount with rate and Weight are displayed (These values are available in web as well as in desktop version).

4.  Cost Price and Gross Margin Rate are displayed only in desktop version.
