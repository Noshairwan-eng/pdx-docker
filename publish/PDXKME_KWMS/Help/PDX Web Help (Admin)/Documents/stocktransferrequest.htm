Stock Transfer Request Inquiry

 

Strategic Stock Requests deals with transferring stock from one warehouse to another. In Parts System a distributor can request to transfer stock to their respective warehouse. There are two main purpose of transferring stock.

1.  Consignment Sales

2.  Reserving Stock for specific distributors

 

Stock Transfer Request Inquiry screen is used to search stock transfer requests with different search criteria. Another main usage of Stock Transfer Request Inquiry screen is to convert Stock Transfer Requests to Stock Transfer Order like Copy to SO from Quotation Inquiry screen.
