How to delete detail line(s) from quotation detail?

There could be two reasons why user wants to delete a line from quotation detail.

1.  Part was added by mistake.

2.  User was just intended to see the pricing information.

 

Detail lines can be deleted from quotation detail via “Delete” button. When a line is deleted from quotation detail two things happen:

1.  Deleted line disappears from quotation detail grid.

2.  All the amounts in footer section get updated (decreased).
