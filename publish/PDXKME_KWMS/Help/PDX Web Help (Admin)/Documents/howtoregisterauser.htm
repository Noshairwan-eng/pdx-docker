How to register a user?

Before registering a user, make sure you have created his record in Parts System using User Creation screen. Now open Register screen.

Enter User ID, User Name, Password, Confirm Password, and Email ID of the user.

Also select DB Code. All this information must match with the record that you have created for new user.

Now press the Register button. Application will match information and then registers the user. 

 

 

After registration, user can successfully access Parts System by user their account.
