Quotation Condition

 

Quotation Condition defines pre sales conditions; set between KME and distributor. Parts System allows user to define these conditions on quotation level. Mainly these conditions affect pricing and allocation. Now we will define the available options on quotation condition screen.

 

Open Quotation menu and select Condition option to access Quotation Condition screen.

The Generate Quotation quick link on Home screen also takes you to Quotation Condition screen.
