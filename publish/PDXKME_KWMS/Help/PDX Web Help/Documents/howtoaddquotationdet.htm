How to add quotation details manually?

 

Follow the below steps to add quotation details manually:

 

1)  Open Quotation detail screen for a specific quotation. Enter Part No and Requested Quantity for all parts that you want to add. Now press the Add button.

 

 

 

 

 

 

 

 

2)  Now press the Calculate button to allocate listed parts.

 

 

3)  Press Update button. A success message is displayed as you update the details.
