What does “Import” button do?

There are two main reasons why Import functionality is needed:

 

1.  Entering parts detail manually in quotation detail screen is quite hectic activity it will take a lot of time if user has to add thousands or even couple of hundred records. 

2.  Most of the customers make their quotations from web portal. Adding parts manually could take a lot of time which could exceed session expiry timeout of KME web, so data will be lost.

 

If you have created a CSV file for quotation detail. By clicking import button you will get a prompt to select CSV file. After selecting a valid CSV file, detail from that file will be populated in detail grid of quotation in just a few seconds.
