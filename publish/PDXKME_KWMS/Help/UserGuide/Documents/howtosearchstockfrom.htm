How to search stock from Quotation Detail screen?

Stock can be searched against any part by clicking on “Stock Search” button. When a user clicks on “Stock Search” button; a new screen titled “Stock Search” will be opened. By entering part number and click on search button will show stock information against specific part.

 

 

The stock search feature enables user to quickly search stock against specific part. E.g. if error “Not enough stock” comes against a specific part then user will be able to see current stock situation and user can modify requested quantity according to availability of the stock.
