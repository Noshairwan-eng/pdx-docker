How to Add Stock Transfer Request Details?

 

Stock Transfer Request Details is the listing of parts that are required in a stock transfer request. After creating an inquiry successfully, its details are added.

Access the Stock Transfer Request Details screen from Strategic Stock Menu.

 

 

Enter Transfer Req No and press Search button to search a transfer request. Form fields show data of provided records.

 

 

You can open the Stock Transfer Request Detail screen from Stock Transfer Request Inquiry screen by searching the record and pressing Detail button.

 

 

The Detail button on Stock Transfer Request Condition screen also open Stock Transfer Request Detail screen for respective record.

 

 

 

Add parts manually or import them from a CSV file. Parts are listed in grid.

 

 

Press the Calculate button for part allocation and cost calculation.

 

 

Now press the Save button to update details
