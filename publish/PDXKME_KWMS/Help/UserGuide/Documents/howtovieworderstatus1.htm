How to View Order Status Inquiry by Part No.?

 

The Order Status Inquiry By Part No screen allows you to search and view order status inquiries by either specifying Part No or select Status of parts.

Open Sales Order menu and select Order Status Inquiry (Part No) option to access the screen.

 

 

To search records of particular part, enter Part No and press Search button. You can provide other criteria to filter records.

 

 

To search inquiries of specific status, select the Status and press Search button. You can provide other criteria to filter records.

 

If invoice data exists for a record, you can view it by selecting the record.
