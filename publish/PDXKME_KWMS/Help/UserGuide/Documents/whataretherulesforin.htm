What are the rules for interchangeability?

There are some conditions by which interchangeability is performed against requested part.

1.  First and most important condition is that part should be a chained part.

2.  If requested part has interchangeable code “016” then it will not be substituted with older parts.

3.  If requested part has interchangeable code “018” then it can be substituted by its immediate old part and then Parts System will see old parts substitution code if that is again “018” then it means requested part can be substituted from one more older part and so on until a “016” coded interchangeable part is reached or chain ends.
