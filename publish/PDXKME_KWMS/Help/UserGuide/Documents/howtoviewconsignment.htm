How to View Consignment Order Inquiries?

 

The Consignment Order Inquiry screen manages list of all inquiries that have been created from Order Condition screen. You can access the records from here and can update them.

Open Consignment menu and select Order Inquiry option to access Consignment Inquiry screen.

 

 

 

To search all inquiries press Search button.

You can filter the records by providing the search criteria in displayed fields and then press Search button; consignment order inquiries that match with provided criteria are listed.

The New button is used to create a new quotation and takes you to Order Condition screen.

 

Click on the inquiry record and you will view options for the selected inquiry.
