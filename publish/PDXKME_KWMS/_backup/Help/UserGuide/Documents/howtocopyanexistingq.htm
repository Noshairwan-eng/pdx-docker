How to Copy an Existing Quotation?

 

You can create a copy of existing quotation. Search and select the record from Quotation Inquiry screen. Press Copy Quotation button, Quotation Copy show options for copying existing quotation.

 

 

The Quotation Copy screen shows options to create copy of selected quotation. Select the option for:

1.  Description of Conversion: The Condition Only option only copy the condition of selected quotation. To copy conditions and details, select All option.

2.  Copy Method: The Copy option creates a copy of existing quotation and in this case Quotation No will remain the same but sub no will be changed for both quotations. The Generation Management option generates a new Quotation No for the copied quotation.

3.  Unit Price: Select unit price option for parts detail.

4.  Way of Giving Line No: Line No can be assigned as of existing quotation and you can also use re-numbering for line no.

 

Press Copy button to create copy of existing inquiry as per specified criteria.

Quotation Condition screen is displayed that show conditions similar to existing quotation. Now assign DB Order No and Save the quotation.

 

 

Press the Detail button to view Quotation Details. The copied quotation is added in the database and is searchable in Quotation Inquiry screen.
