Which errors can occur while calculating quotation detail?

There could be multiple errors which can occur, most common errors are:

1.  Part No Not Found (When invalid part is entered)

2.  In case of EO or SE duplicate parts are not allowed.

3.  Consignment parts can’t be ordered through quotation.

4.  On web portal if requested quantity exceeds available stock quantity then “Stock quantity is not available” error will occur.

5.  In case of Emergency Order if user requests quantity more than machine quantity then error will occur.

6.  If not supply parts are requested then error will occur.
