How to Add Consignment Order Details?

 

Consignment Order Details is the listing of parts that are required in a consignment order. After creating an inquiry successfully, its details are added.

Access the Consignment Order Details screen from Consignment Menu.

 

 

 

You can also access the Consignment Order Detail screen from Consignment Inquiry screen.

 

 

 

Enter Order No and press Search button to search a consignment inquiry. Form fields show data of provided record.

 

 

Add parts manually or import them from a CSV file. Parts are listed in grid.

 

 

Press the Calculate button for part allocation and cost calculation.
