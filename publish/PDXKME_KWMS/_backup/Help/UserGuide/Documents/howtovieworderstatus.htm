How to View Order Status Inquiry by Order No.?

 

The Order Status Inquiry By Order No screen allows you to search and view order status inquiries by order no.

Open Sales Order menu and select Order Status Inquiry (Order No) option to access the screen.

 

 

You can search records by providing Sale Order No or Distributer Order No. Select the option from Search By dropdown and enter Order No. Now press the Search button; part records are listed.

Select a record from list to view its Invoice Data.
