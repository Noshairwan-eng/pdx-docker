How to Register Consignment Order?

 

The Order Condition screen is used to create/register a new consignment order.

Open Consignment menu and select Order Condition option to access the screen.

 

 

 

 

Select Order Type from the dropdown. You can also provide Requested Delivery Date along with Order Recv Date, Comments, and Memo. Press Save button; Consignment order will be registered and an Order No will be assigned to it.

 

 

 

The newly registered order is accessible from Order Inquiry screen.
