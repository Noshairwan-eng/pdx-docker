How to Delete a Quotation Inquiry?

 

You can only delete a quotation if its status is “Cancelled”. Purpose of deleting a quotation is that a distributor does not want to see unnecessary data in search results. There is no use of keeping “Cancelled” quotations in system.

To delete a quotation inquiry, you need to:

1.  Search cancelled quotation by certain criteria.

2.  Select quotation line to delete.

3.  Press Delete button, a confirmation message will appear; on confirm the quotation will be deleted.
