How to Register Quotation?

 

The Quotation Condition screen is used to create/register a new quotation.

Open Quotation menu and select Condition option to access Quotation Condition screen.

The Generate Quotation quick link on Home screen also takes you to Quotation Condition screen.

 

 

 

Enter DB Order No. Also select the Order Type.  All other mandatory fields will be listed on the basis of selected Order Type.

 

 

 

Order Type:

For all order types, you only need to mention DB Order No and Order Type. All other fields are selected on the basis of Order Type selection.

Press the Save button to register quotation. Quotation is saved and a system generated Quotation No is assigned to it. The newly created quotation is now available on Quotation Inquiry screen.

 

 

If you try to save the quotation but some mandatory field is missing then the required fields are highlighted.

 

Model Information:

For some orders, it is mandatory to provide model information.

Press the Save button to register quotation. Quotation is saved and a system generated Quotation No is assigned to it. The newly created quotation is now available on Quotation Inquiry screen.
