How to Register Stock Transfer Request?

 

The Stock Transfer Request Condition screen is used to create/register a new quotation for transferring stock from one warehouse to another.

Open Strategic Stock menu and select Stock Transfer Request Condition option to access the screen.

 

 

 

The DB Order No and Order Type are mandatory fields. Provide the data and press Save button to register request.

 

 

As you save the condition after providing mandatory data, a success message is displayed. Transfer Req No and Recommendation No are also generated for the new record.
