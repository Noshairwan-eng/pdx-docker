Quotation Inquiry

 

The Quotation Inquiry screen manages list of all inquiries that have been created. From this screen, user is able to search Quotations with different search criteria. User can perform different operations on these searched quotations.

To access the screen, open Quotation menu and select Quotation Inquiry option.

 

 

The Quotation Inquiry screen provides the following features:

 

1.  View quotation/s that have been created

2.  Delete cancelled quotation

3.  Print a quotation for which detail exists

4.  Make a copy of existing quotation

5.  View conditions of quotation

6.  View details of quotation

7.  Convert the quotation into sales order

8.  View the conditions of sales order against selected quotation
