How to make a Sales Order from Quotation?

 

Quotation can be converted to Sales Order. This feature is useful to register Sales in Parts System when you are sure about a quotation. After creating a quotation, you have clear picture of the stock you are going to be allocated and prices incur. But at this point sales are not registered which means stock is not actually allocated. When you create Sales Order (Copy to SO) from quotation then actual stock will be allocated to that Sales Order and sales in Parts System will be registered.

To copy a quotation to Sales Order, following rules must be fulfilled.

1.  Quotation should have Confirmed status.

2.  Quotation is created using latest list price.

 

From the searched records on Quotation Inquiry screen, select a Confirmed quotation and press Copy to SO button. The Copy To Sales Order screen is displayed.

 

 

The screen shows following fields:

 

Ø Quotation No: Shows the Quotation No. of selected inquiry.

Ø Back Order Direct Shipment Code

Ø Partial Shipment Allowed: This option indicates that whether an order should be processed in partial or as a whole w.r.t availability of stock. If this option is checked then parts in an order against which stock is allocated can be invoiced but the rest of the parts which were not processed due to any reason can be processed later on. If you uncheck this option, Parts System will restrict the user to pick the parts from stock until all the parts are allocated stock completely.

 

 

By pressing the Copy button quotation becomes a sales order and a message is displayed.

 

 

The following process will be performed at backend.

ü Allocate the stock from warehouse and available on order

ü Create picking instruction for allocated parts

ü And, perform packing process for picked parts

The status of respective record will be changed to Copy to Sales Order
