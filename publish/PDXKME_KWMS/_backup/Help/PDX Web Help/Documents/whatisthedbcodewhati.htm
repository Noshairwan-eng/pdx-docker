What is the DB code? What is its purpose?

All the KME distributors are assigned unique codes which are called distributor codes. When a user signs in, this field gets populated automatically with the distributor code from which signed in user belongs to (This field remains disable for distributors). But in desktop version this list may have multiple customer codes. All customers are divided among different KME users, so when a KME user will log in he will be able to see his assigned customer codes in this field.
