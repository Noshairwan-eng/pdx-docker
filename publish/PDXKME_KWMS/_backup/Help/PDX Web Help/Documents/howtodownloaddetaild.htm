How to download detail data from quotation?

Detail data can be downloaded for the purpose of record keeping by the distributor. Later on user will be able to know that at which price they received those parts, how much tax incur and how much discount KME offered them etc.

If calculated detail exists against quotation, only then data can be exported. Detail data can be exported in CSV and XLS formats.
