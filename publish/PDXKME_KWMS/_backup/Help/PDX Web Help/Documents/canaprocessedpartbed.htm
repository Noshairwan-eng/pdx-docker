Can a processed part be different from requested part?

Yes, processed part can be different from requested part. It depends on multiple different factors

1.  If requested part is part of some chain. (If requested part is not a chained part then it can’t be substituted with any other part)

2.  Substitution code (selected on quotation condition).

3.  Interchangeable Code (If it is backward compatible or not).

4.  Availability of stock.
