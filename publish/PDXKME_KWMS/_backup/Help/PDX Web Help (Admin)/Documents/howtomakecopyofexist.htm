How to Make Copy of Existing Quotation?

 

You can create a copy of existing quotation. Search and select the record from Quotation Inquiry screen. Press Copy Quotation button, Quotation Copy show options for copying existing quotation.

 

Copy Quotation feature is useful when you want to make a quotation that has same condition or details as in some existing quotation. So, instead of making a new quotation from scratch, use copy option to copy condition and detail (or only condition) information from previously created quotation and make a new quotation.

 

Note: Quotations with only latest list price can be copied otherwise error message is displayed that says “that quotation created with old list price cannot be copied, revise it with latest list price”.

 

Following are the steps to copy a quotation:

 

1.  Search and select an existing quotation from Quotation Inquiry screen

2.  Press Copy Quotation button. Quotation Copy screen is displayed.

 

 

The Quotation Copy screen shows the following fields:

 

Ø Reserve: When this checkbox is checked then requested quantity will be reserved from available stock. Purpose of reserving stock on quotation stage is that the customer already occupies the stock and while converting quotation to the sales order chances are there that he will get the reserved stock.  

Note: Reserved stock is release when quotation gets expired.

Ø Orig Quotation No: Number of Quotation from which new quotation will be copied.

Ø Quotation No: Newly created Quotation Number.

Ø Description of Conversion: Description of conversion defines which part of previous quotation should be copied to new quotation.

o  All:  Condition and Detail information will be copied.

o  Condition Only: As the name suggests this will only copy condition information.

Ø Copy Method: Method to be used in the copying process

o  Copy: This method will generate new quotation number. It is useful when you want to create a new quotation having similar Condition & Detail as of existing quotation.

o  Generation Management: This method will generate same quotation number with incremented subordinate number. It is useful when you want to revise quotation and make some modifications in condition or detail and also want to keep copy of previous quotation.

Ø Unit Price: There are also two methods of calculating Unit price for new quotation.

o  Use Current Price: This option will apply same prices in new quotation as in old quotation.

o  Recalculate: This option will apply latest prices for new quotation.

Ø Way of giving Line Number:  There are two ways of giving line numbers to the new quotation.

o  Use Current No: If the older quotation has some deleted lines from detail (Its line numbers are no more in sequential manner). Then newly copied quotation will have same non sequential line numbers.

o  Renumbering: This method will re-number all the detail lines of newly copied quotation in sequential manner.

 

Pressing the Copy button will create new quotation with the settings made in steps mentioned above.

If the selected quotation contains error parts or does not contain any detail then an error message is displayed upon pressing the Copy button.
