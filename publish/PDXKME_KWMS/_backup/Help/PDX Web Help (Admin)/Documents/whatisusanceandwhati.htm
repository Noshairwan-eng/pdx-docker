What is “Usance” and what is its usage?

Usance is the grace time given to the distributor to clear its payment after invoicing. In Parts System, usance is set per distributor per order type. There can be five values against usance 0, 15, 30, 60 and 90.

 

(Note: If a payment term is selected in which payment date is mentioned then usance will apply after that time period).
