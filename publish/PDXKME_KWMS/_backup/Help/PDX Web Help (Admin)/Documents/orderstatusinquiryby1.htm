Order Status Inquiry by Part No.

 

The Order Status Inquiry By Part No screen allows you to search and view order status inquiries by either specifying Part No or select Status of parts.

Open Sales Order menu and select Order Status Inquiry (Part No) option to access the screen.

 

 

You can search records by providing part number or status.
