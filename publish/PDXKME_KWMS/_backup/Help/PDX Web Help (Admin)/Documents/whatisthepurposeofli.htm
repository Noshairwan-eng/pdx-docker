What is the purpose of “Line” and “Sub Line No” fields in detail grid?

Line No defines how many parts are added by a user in quotation detail screen. When a part is substituted by single part then line number will be incremented but sub line number will be 0. But if a requested part is substituted by multiple parts then line number will be the same but sub line number will be incremented on new line and each line defines how much quantity is accepted against each chained part.
