What is “Status” and what each status means?

Status of a quotation defines at which state currently quotation is. There are four statuses a quotation can have:

1.  In-Process: When quotation is in edit mode anything can be changed which is subject to change.

2.  Confirmed: When quotation is confirmed it means that user has completed making changes in quotation. If a quotation is in confirmed status and you want to make some changes in it then you have the option to update its status back to In-Process and make required changes and again confirm it. Confirmed quotation can be copied to sales order.

3.  Copy To SO: This status defines that searched quotation has been copied to sales order and no changes can be made in this quotation.

4.  Cancelled: Quotation can only be cancelled if no detail exists against it. A cancelled quotation can’t be edited again.
