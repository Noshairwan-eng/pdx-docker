Creating New Quotation

 

Select DB Code and Order Type. Also, enter DB Order No. All other mandatory fields will be listed on the basis of selected DB Code and Order Type.

 

 

 

In certain cases, by selecting DB Code and Order Type all mandatory fields are not filled up. In that case you have to select data for mandatory fields to register quotation successfully.

 

 

Press the Save button to register quotation. Quotation is saved and a system generated Quotation No is assigned to it. The newly created quotation is now available on Quotation Inquiry screen.

 

 

For some order types, it is mandatory to provide model information.

In that case at least one model must be specified after providing DB Order No and Order Type.

 

 

 

 

If you try to save the quotation but some mandatory fields are missing then the required fields are highlighted.
