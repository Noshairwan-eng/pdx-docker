Order Status Inquiry by Order No.

 

The Order Status Inquiry By Order No screen allows you to search and view order status inquiries by order no. It shows the information of a sales order, line by line for each part and also shows part’s stage; from order quantity to invoice generated.

To access the screen, open Sales Order menu and select Order Status Inquiry (Order No) option.
