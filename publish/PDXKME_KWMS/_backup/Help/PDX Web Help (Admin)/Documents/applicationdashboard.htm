Application Dashboard

 

Upon accessing Parts System application dashboard is displayed.

The Menu bar shows all modules. Move the cursor over a module to view its available sub options.

Quick links provides ease to access main features of the application.

 

 

 

The Dashboard also provides feature to download the latest price list of all parts. It is available is CSV format.

Click on Download List Price link. File starts to download and display success message after successful completion. Open the view and view price information of all parts.
